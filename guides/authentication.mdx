# Authentication

The Influship API uses API key authentication. All requests must include your API key in the `X-API-Key` header.

## Getting Your API Key

1. Sign up at [dashboard.influship.com](https://dashboard.influship.com)
2. Navigate to **API Keys** section
3. Click **Generate New Key**
4. Copy and store your key securely

<Warning>
  Your API key is displayed only once. Store it securely and never share it
  publicly.
</Warning>

## Using Your API Key

Include your API key in the `X-API-Key` header for all requests:

<CodeGroup>
```bash cURL
curl -H "X-API-Key: YOUR_API_KEY" \
  https://api.influship.com/health
```

```javascript Node.js
const response = await fetch('https://api.influship.com/health', {
  headers: {
    'X-API-Key': process.env.INFLUSHIP_API_KEY,
  },
});
```

```python Python
import requests

response = requests.get(
  'https://api.influship.com/health',
  headers={'X-API-Key': 'YOUR_API_KEY'}
)
```

</CodeGroup>

## Error Responses

### Invalid API Key

<ResponseExample>
```json 401 Unauthorized
{
  "error": {
    "code": "invalid_api_key",
    "message": "The provided API key is invalid or expired"
  }
}
```
</ResponseExample>

### Missing API Key

<ResponseExample>
```json 401 Unauthorized
{
  "error": {
    "code": "missing_authorization",
    "message": "Authorization header is required"
  }
}
```
</ResponseExample>

## Best Practices

- Store your API key as an environment variable
- Never commit API keys to version control
- Use different keys for development and production
- Rotate keys regularly for security
