## üü° Partially Working

### Creator Search
- **Endpoint**: `POST /v1/search`
- **Status**: ‚ö†Ô∏è **Partially Working**
- **Description**: AI-powered creator search with natural language queries
- **Dependencies**: 
  - Pinecone vector database with creator embeddings
  - OpenAI API for query embedding and reranking
  - Supabase database with creator data and views
- **What Works**: Code structure, validation, rate limiting, billing
- **What's Broken**: Returns empty results or errors without proper database setup
- **Notes**: `live` mode is not implemented

### Creator Enrichment (Legacy Endpoints)
- **Endpoints**: 
  - `GET /v1/creators/{id}`
  - `GET /v1/creators/{platform}/{username}`
  - `GET /v1/profiles/{platform}/{username}`
- **Status**: ‚ö†Ô∏è **Partially Working**
- **Description**: Get detailed creator profiles and platform data
- **Dependencies**: Supabase views (`api_creators_lite`, `api_instagram_profiles_lite`, etc.)
- **What Works**: Code structure, validation, authentication
- **What's Broken**: Depends on database views that may not exist
- **Notes**: `live` mode is not implemented

### Lookalike Search
- **Endpoint**: `POST /v1/lookalike`
- **Status**: ‚ö†Ô∏è **Partially Working**
- **Description**: Find similar creators based on seed creators
- **Dependencies**: 
  - Pinecone with populated creator embeddings
  - Supabase creator data
- **What Works**: Code structure, vector similarity computation
- **What's Broken**: Requires Pinecone setup and populated data
- **Notes**: `live` mode is not implemented

### Match Scoring
- **Endpoint**: `POST /v1/match`
- **Status**: ‚ö†Ô∏è **Partially Working**
- **Description**: Score creators against campaign requirements
- **Dependencies**: Database content for creator documents
- **What Works**: Returns computed scores using available creator docs
- **What's Broken**: Depends on database content quality
- **Notes**: `live` mode is not implemented
