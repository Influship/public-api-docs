## Quick Start

<Steps>
<Step title="Get API key">
  Sign up at [dashboard.influship.com](https://dashboard.influship.com)
</Step>

<Step title='Test API'>
  ```bash curl -H "X-API-Key: YOUR_KEY" https://api.influship.com/health ```
</Step>

<Step title="Search creators">
  ```bash
  curl -X POST https://api.influship.com/v1/search \
    -H "X-API-Key: YOUR_KEY" \
    -H "Content-Type: application/json" \
    -d '{"query": "fitness influencers"}'
  ```
</Step>
</Steps>
