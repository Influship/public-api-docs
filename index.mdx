## Quick Start

Get started with the Influship API in minutes:

<Steps>
<Step title="Get your API key">
  Sign up at the [Influship Dashboard](https://dashboard.influship.com) to get your API key.
</Step>

<Step title="Make your first request">
  Test the API with a simple health check:
  
  ```bash
  curl -H "X-API-Key: YOUR_API_KEY" \
    https://api.influship.com/health
  ```
</Step>

<Step title="Search for creators">
  Find influencers using natural language:
  
  ```bash
  curl -X POST https://api.influship.com/v1/search \
    -H "X-API-Key: YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"query": "fitness influencers with 100k+ followers"}'
  ```
</Step>
</Steps>
