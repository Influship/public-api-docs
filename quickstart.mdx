# Quickstart

Get started with the Influship API in just a few minutes. This guide will help you make your first API call and understand the basics.

## Prerequisites

- An API key from [elliot@influship.com](mailto:elliot@influship.com)
- A tool to make HTTP requests (curl, Postman, or your preferred client)

## Step 1: Get Your API Key

<Steps>
<Step title="Request Access">
  Email [elliot@influship.com](mailto:elliot@influship.com) to get your API key.
</Step>

<Step title="Store Securely">
  Copy your API key and store it as an environment variable:
  ```bash
  export INFLUSHIP_API_KEY="your_api_key_here"
  ```
</Step>
</Steps>

## Step 2: Test the API

Start with a simple health check:

<RequestExample>
```bash cURL
curl -X GET 'https://api.influship.com/health' \
  -H 'X-API-Key: YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "ok": true,
  "timestamp": "2024-01-15T10:30:00.000Z"
}
```
</ResponseExample>

<Check>
Great! Your API key is working. Now let's find some influencers.
</Check>

## Step 3: Find Your First Influencers

Search for influencers using natural language - this is where the magic happens:

<RequestExample>
```bash cURL
curl -X POST 'https://api.influship.com/v1/search' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "query": "fitness influencers with high engagement",
    "limit": 3
  }'
```
</RequestExample>

<ResponseExample>
```json Search Results
{
  "items": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "name": "Sarah Johnson",

```
</ResponseExample>

## Step 4: Get Detailed Creator Information

Now let's get more details about the creators you found:

<RequestExample>
```bash cURL
curl -X POST 'https://api.influship.com/v1/creators' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "creator_ids": ["123e4567-e89b-12d3-a456-426614174000"],
    "include_profiles": true,
    "profile_mode": "detailed"
  }'
```
</RequestExample>
